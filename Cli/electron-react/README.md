## 开始

### 实现
- 使用 electron + create-react-app + antd 封装而成的跨平台桌面端脚手架
- 本机系统环境
  - macOS 10.13.1
  - nodejs 10.8.0
  - yarn 1.12.3 [registry: https://registry.npm.taobao.org]

### 构建提供的功能(Webpack)
- antd, 自助主题配置
- 全局 LESS 变量配置
- 支持 ES7 装饰器语法
- 配置别名 ALIAS

## 启动
``` bash
yarn run start
```

## 打包构建

### mac

``` bash
yarn run build:mac
```

### win

``` bash
yarn run build:win
```
