该脚手架为 electron 脚手架, 功能包含:
- antd 能够配置主题变量 `~/config/custom.config.js`
- css 预编译器 less, 支持 styled-components
- polyfill 支持 ie 9
- 支持 ES7 装饰符号
- 支持 mobx react-router-dom

## 运行
``` bash
# 开启 react 服务
npm run start

# 开启 electron 服务
npm run electron
```

## 打包
``` bash
# 打包 react 项目
npm run build

# 打包 electron
npm run package
```
